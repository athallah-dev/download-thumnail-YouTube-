YouTube Thumbnail Downloader 🚀

Welcome to the YouTube Thumbnail Downloader tool.

This tool allows you to extract and download high-quality thumbnails from any YouTube video instantly.

How to use:
1. Paste any YouTube video link into the input field below.
2. The thumbnails will automatically appear without clicking any button.
3. Click the "Download Image" button to open the image.
4. On mobile devices, long-press the image and select "Save Image".

This tool supports standard videos, short links, and YouTube Shorts.
